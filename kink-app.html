Project information

Overview

I created a responsive beauty app. The apps goal is "give users access to quality information"

Design Process

Focusing on UI for this project, the steps i covered in my design process include: user stories, personas, user flows, sketching, wireframing, style guide and prototyping.

Define

The first steps in my process involved defining and answering the 5 W questions.

Empathize

After defining the scope, the next step was creating user stories and persona to better understand my users' needs and goals.

Ideate

I then figured out the essential features the app would need to make it successful and useful. I also created a user flow diagram to map out these features and see what kind of pages would be necessary for the app.

Design

Next, I sketched low-fidelity wireframes with pencil and paper. I then moved on to figma for my mid-fidelity wireframes, where i used a 12-column grid to create the initial layouts for mobile, tablet, and desktop devices.
